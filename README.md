## 项目简介

面向K12的AI智能作业批改系统，覆盖数学、语文、英语三大学科，实现：

- 🔍 **多学科自动批改**：客观题+主观题全覆盖
- 📊 **知识点薄弱诊断**：知识图谱驱动，定位到具体知识点链路
- 💬 **个性化评语生成**：基于学情档案，生成"像班主任写的"评语
- 📈 **学情趋势追踪**：连续作业形成画像，自动预警成绩波动

## 快速启动

### 环境要求

- Python 3.9+
- OpenAI API Key（或兼容API）

### 安装

```bash
cd 希沃
pip install -r requirements.txt
```

### 运行

```bash
streamlit run demo/app.py
```

浏览器自动打开 `http://localhost:8501`

### 配置API

1. 在左侧边栏输入 API Key
2. 如使用国内模型，填写 Base URL（如 `https://dashscope.aliyuncs.com/compatible-mode/v1`）
3. 选择对应模型（如 `qwen-vl-max`）

## 核心功能演示

### 1. 作业批改

选择学科 → 输入题目和学生答案 → 点击"开始批改"

系统返回：
- 分数（0-10）
- 对错判断
- 错误类型归因
- 正确答案/解题过程
- 涉及知识点
- 个性化改进建议

### 2. 学情诊断

自动基于批改结果生成：
- 知识点掌握度雷达图
- 错误类型分布饼图
- 知识图谱薄弱链路追踪
- 个性化教师评语

### 3. 示例演示

无需API Key即可体验：
- 数学：方程求解、因式分解、几何证明
- 语文：阅读理解赏析、造句、作文
- 英语：翻译、时态、不规则动词

## 技术亮点

| 亮点 | 说明 |
|------|------|
| 诊断式批改 | 不只判对错，定位"哪一步出错+为什么错" |
| 知识图谱 | 200+知识点节点，支持前置依赖追溯 |
| 多学科差异化 | 数学分步验证、语文维度评分、英语语法标注 |
| 个性化评语 | 基于历史学情生成，不是模板套话 |
| 教师可调 | AI先批→教师复核→持续学习 |
